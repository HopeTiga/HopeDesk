### WebRTC 远程桌面操控系统

基于 `LibWebRTC`、`Openh264`、`Boost` 和 `Qt` 实现的 P2P 远程桌面控制方案


## 核心组件

### 1. 信令服务器
- **技术栈**: LibWebRTC + Boost
- **功能**:
  - 处理客户端注册/发现
  - 交换 SDP 和 ICE 候选
  - 状态管理

### 2. 被控端 (视频源)
- **技术栈**: Qt + LibWebRTC
- **功能**:
  - 屏幕捕获 (DXGI/Qt)
  - H264 实时编码
  - WebRTC 视频流传输
  - 接收控制指令

### 3. 操控端
- **技术栈**: Qt + LibWebRTC
- **功能**:
  - 视频流解码渲染
  - 输入事件转发
  - 信令交互
