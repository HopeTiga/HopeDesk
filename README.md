### WebRTC 远程桌面操控系统
基于 `LibWebRTC`、`Boost` 和 `Qt` 实现的 P2P 远程桌面控制方案

## 核心组件

### 1. 信令服务器
- **技术栈**: Boost (网络通信) + WebSocket/HTTP
- **功能**:
  - 处理客户端注册/发现
  - 交换 SDP 和 ICE 候选
  - 房间管理和状态维护
  - 连接建立协调

### 2. 被控端 (视频源)
- **技术栈**: Qt + LibWebRTC
- **功能**:
  - 屏幕捕获 (DXGI/Qt ScreenCapture)
  - LibWebRTC 内置视频编码 (VP8/VP9/H264)
  - WebRTC P2P 连接管理
  - RTP 视频流传输
  - 接收并执行远程控制指令
  - ICE 连接建立与维护

### 3. 操控端 (控制客户端)
- **技术栈**: Qt + LibWebRTC  
- **功能**:
  - LibWebRTC 视频流接收与解码
  - Qt Widget 视频渲染显示
  - 本地输入事件采集与传输
  - WebRTC DataChannel 控制指令发送
  - 信令交互与连接管理

## 技术优势

- **纯 P2P 通信**: 基于 LibWebRTC 实现真正的点对点连接，减少服务器压力
- **跨平台支持**: Qt 框架保证 Windows/Linux/macOS 兼容性  
- **低延迟**: 硬件编码 + UDP 传输 + P2P 直连
- **高可靠**: 完整的 WebRTC 协议栈，包含网络穿透和错误恢复机制
